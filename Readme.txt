Как развернуть приложение:

1. запустить Docker

2. скачать/скопировать на диск папку Hackaton2, в которой:

    - папка App (с файлами приложения):
        requirements.txt - список пакетов для установки при создании образа docker
        pipe.pkl - пайплайн
        main.py - скрипт самого приложения (api)


    - Dockerfile
    - Readme - данная инструкция

3. в Git Bash перейти в папку Hackaton2

4. выполнить команду:
    docker build -t hackatonim .
    (сгенерируется образ hackatonim со всеми нужными библиотеками)

5. Запуск контейнера по созданному образу hachatonim:

запуск контейнера с нормальным именем (hackatonapp) и указанием портов!:
    docker run -p 8501:8501 --name=hackatonapp hackatonim

остановка контейнера
    docker stop hackatonapp

проверить, что контейнер запущен/скопировать id:
    docker ps

зайти внутрь контейнера (bash)
    docker exec -it <id_контейнера> bash

После запуска контейнера наше приложение становится доступно по адресу:
    localhost:8501 или http://127.0.0.1:8501/
